This game is made with OVERLORD ENGINE
=======================================

Controls
=========
Xbox Controllers (player 1-4):
------------------------------- 
	- Move: DPad Buttons
	- Drop Bomb: A-Button
	- Punch Bomb: X-Button

KeyBoard:
---------
player1: 
	- Move: Numeric Keypad 8,4,5,6
	- Drop Bomb: Keypad 7
	- Punch Bomb: Keypad 9
player2: 
	- Move: w, a, s, d
	- Drop Bomb: q
	- Punch Bomb: e
player3: 
	- Move: y, g, h, j
	- Drop Bomb: t
	- Punch Bomb: u
player4: 
	- Move: arrow keys
	- Drop Bomb: ctrl
	- Punch Bomb: right shift
